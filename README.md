# wows-fast-stats
## About
This is the fast XVM for World of Warships.

## Development
### Requirements
- make >= 3.81
- go >= 1.20
- npm >= 8.19.2

### Run locally
```
$ git clone git@github.com:tona0516/wows-fast-stats.git && cd wows-fast-stats/
$ make setup
$ make dev
```

## Build executable
```
$ make build
$ ls build/bin
wows-fast-stats-0.3.0.exe*
```
